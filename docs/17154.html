Overview

# NeuroMM-2026 挑战赛 — 赛道 2：NMM-Context-IED 后期评估 欢迎参加 **NeuroMM-2026 多模态神经生理智能挑战赛的赛道 2（NMM-Context-IED）**。 竞争阶段已结束。该平台在有限时间内对外开放，仅供进行消融研究；这些后续结果将不会被纳入官方排行榜排名。 ### NMM-Context-IED - 主要指标：AUPRC - 重点：通过同步视觉上下文特征进行伪影抑制 结合同步上下文特征的视觉增强检测。 ## 任务 二分类：对于每个样本，预测其包含癫痫**尖波**（正类，标签 `1`）与**非尖波**（负类，标签 `0`）的概率。赛道 2 **鼓励团队将预提取的同步视觉上下文特征与生理信号（脑电图/肌电图/心电图）相结合**，以抑制伪影并提高相比仅信号检测的鲁棒性。 ## 数据 从以下地址申请访问权限并下载数据集： > <https://huggingface.co/datasets/NeuroMM/NeuroMM-2026> 数据集包含**训练集**、**验证集**和**候选集**（包含赛道 2 测试样本与干扰样本的混合）。候选集故意设计得大于官方赛道 2 集合，**不会告诉您哪些样本被评分** — 请对所有已发布的候选集中的样本 ID 进行预测并提交；评分器将自动仅保留官方隐藏的赛道 2 样本并忽略其余部分。 ## 注册规则 - 一个人只能参加**同一支团队**。 - 一支团队可参加多个赛道，但**每个赛道仅限一名团队成员的账户**进行提交。 - 在 **Codabench 注册所用的电子邮件必须与** HuggingFace 数据集访问请求表单中填写的电子邮件**一致**；否则注册**将不被批准**。所有注册均由组织者手动审核。 ## 提交 本竞赛仅接受**预测结果**。您需上传单个预测文件（打包为 ZIP 格式） — 不需要提交代码或模型。详见**评估**页面了解确切格式。 ## 排名 提交作品按 **AUPRC**（精确率-召回率曲线下面积）排名。这是**唯一**的指标 — 不计算、报告或显示其他指标。

Evaluation

# 评估 ## 指标 **AUPRC** — 精准率-召回率曲线下面积（也称为平均精准率）， 以 `label = 1`（尖峰）作为正类计算。这是**唯一的** 指标；不产生、报告或显示其他指标。数值越高越好。 ## 提交格式 提交一个**ZIP**文件，其中恰好包含一个名为**`submission.csv`**的文件： ``` sample_submission.zip └── submission.csv ``` > 只接受文件名 `submission.csv`。任何其他名称 > （例如 `prediction.csv`）都将导致提交失败。 `submission.csv` 有两列： ```csv sample_id,prediction NMM26-XXXXXXXX,0.0123 NMM26-YYYYYYYY,0.9871 NMM26-ZZZZZZZZ,0.5040 ``` - `sample_id` — 候选集中给出的完全相同的id。 - `prediction` — 一个实值**概率/分数，表示样本是 尖峰的概率**（正类）。任何单调分数都适用于AUPRC；不要求 值被校准（推荐使用 `[0, 1]` 范围）。 ### 重要事项 - 为发布的候选集中的**每个**id预测，并包含所有预测。 **不会告知**哪些id是官方第2赛道样本。 - 评分器只保留隐藏的官方第2赛道id，并**静默 忽略所有其他行**（干扰项/其他赛道）。额外的行是 可以接受的。 - 如果任何官方第2赛道样本没有预测、预测 不是数值、`sample_id` 重复，或文件 名不是 `submission.csv`，提交**失败**。

Terms

# 条款和条件 参与NeuroMM-2026挑战赛（赛道2：NMM-Context-IED）即表示您同意： - 提供的数据**仅限**用于研究和参加本挑战赛。禁止数据的重新分发或任何商业/未授权使用。 - 使用自动化机器学习方法进行预测。通过人工检查生理信号对样本进行手动标注**不是**可接受的方法，违反了本挑战赛的初衷。 - 不尝试对样本进行去匿名处理、识别哪些候选ID是评分官方样本，或以其他方式对隐藏标签进行逆向工程。 - 进行合理数量的提交（执行每日上限）。禁止滥用评估服务来探测隐藏标签。 组织者可在必要时更新这些规则和竞赛时间表，并保留取消违反这些条款的提交资格的权利。 该数据集按照NeuroMM-2026数据使用政策提供（仅限学术研究和挑战赛参与）。
