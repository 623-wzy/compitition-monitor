Aufgabenbeschreibung

# TreeVision 挑战赛 TreeVision 挑战赛致力于使用现代计算机视觉方法在近地面拍摄的图像中自动识别树木。 参赛者需要开发目标检测模型，能够在不同的环境条件、照明情况、视角和尺寸下可靠地定位树木。 该挑战赛旨在促进开发强大而高效的目标检测方法，用于环境监测、林业应用和实际计算机视觉场景。

Evaluation

# 评估 评估在未知的测试数据集上进行。目标是评估预测的边界框与真实标注的质量。 ## 真实值与预测的匹配 首先，对于每个真实边界框，确定相同类别中交并比（IoU）最高的预测。 * 每个预测最多只能分配给一个真实框。 * 对于每个真实框，选择未被使用且IoU最高的预测。 * 不同类别的预测不被考虑。 ## 交并比（IoU） 交并比（IoU）是衡量预测边界框与对应真实边界框重叠程度的指标。 它计算为交集面积与并集面积的比值： ```text IoU = 交集面积 / 并集面积 ``` 其中： * IoU = 1.0 边界框完全匹配 * IoU = 0.0 边界框不重叠 ## 预测分类 当最佳IoU超过设定的阈值时，该分配被视为真正例（TP）： ```text IoU ≥ 0.5 ``` 否则，该真实框被计为假负例（FN）。 无法分配给任何真实框的预测被视为假正例（FP）。 ## 评估指标计算 基于聚合的真正例、假正例和假负例的值，计算以下指标： ### 精准率 精准率表示所有预测对象中正确预测的比例。 ```text 精准率 = TP / (TP + FP) ``` ### 召回率 召回率表示所有实际存在的对象中被正确识别的比例。 ```text 召回率 = TP / (TP + FN) ``` ### F1分数 F1分数是精准率和召回率的调和平均值。 ```text F1 = 2 · 精准率 · 召回率 / (精准率 + 召回率) ``` ## 边界框IoU计算 另外，对于每个真实框，保存找到的最高IoU值， 无论其是否达到0.5的阈值。 随后从所有保存的IoU值计算平均边界框IoU： ```text 平均边界框IoU = 所有最佳IoU值的平均值 ``` 该指标描述真实边界框与相应最佳预测之间的平均重叠程度。 ## 评分 最终评估基于平均边界框IoU。 * 平均边界框IoU为0.7或更高时，获得最高分数100分。 * 对于低于0.7的值，在0到100分之间进行线性缩放。 ```text 平均边界框IoU ≥ 0.7 → 100分 平均边界框IoU < 0.7 → (平均边界框IoU / 0.7) · 100 ```
