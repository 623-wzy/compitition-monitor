Overview

<div align="center"> # 使用多模态影像数据的二尖瓣解剖分析 ## MICCAI 2026挑战赛 - 医学世界模型国际研讨会(MWM) </div> 二尖瓣评估在结构性心脏病的诊断、治疗规划和术中引导中起着核心作用。在临床实践中，医生依赖具有非常不同特征的多模态影像数据，包括用于术前几何结构的心脏CT、用于术中解剖参考的3D经食管超声心动图，以及用于直接手术观察的选定术中帧。然而，跨这些模态的手动分析仍然劳动密集且受观察者间变异性的影响。 MVAA挑战赛旨在通过提供跨多种影像模态和手术阶段的统一基准来推进自动化二尖瓣分析。鼓励参赛者开发稳健的分割和解剖分析方法，这些方法在异构数据源上保持一致，并支持临床意义的下游应用。 ## 挑战赛目标 开发用于跨CT、3D TEE和术中图像帧的多模态二尖瓣解剖分析的自动化方法，评估重点关注分割质量、结构一致性和临床相关性。 **具体目标：** - 建立涵盖术前、术中和手术影像数据的二尖瓣分析多模态基准。 - 评估算法在多样化影像特征和临床条件下的准确分割和解剖理解能力。 - 鼓励开发在模态间具有良好泛化性能同时保留二尖瓣连贯结构表示的模型。 - 促进AI研究人员、影像科学家和心脏介入临床专家之间的协作。 ## 资源 - **挑战赛网站:** https://mwm2026.github.io/mvaa - **数据集(训练与验证) - 链接1:** 任务1和任务2的完整数据，加上任务3的标注数据: https://drive.google.com/file/d/14WneBUBZ1X4p69tRdRzximNb0IsWuh2B/view?usp=sharing - **数据集(训练与验证) - 链接2:** 任务3的未标注数据: https://drive.google.com/file/d/1K4vFo-4pdL1_xt-8msO-32bc2aEdJ7wS/view?usp=sharing - **基线代码:** https://github.com/db0725/MVAA - **联系方式:** dbd2002@foxmail.com

Core Task

# 核心任务 MVAA挑战赛旨在反映跨多种成像模式的真实临床工作流程。该挑战赛分为三个主要分割任务。 ### 参赛和模型提交要求 参赛者需要完成并提交全部三个任务的结果。最终排名将基于三个任务的综合性能确定；因此，仅提交一个或两个任务的结果不足以进行最终排名。 但是，参赛者无需使用单一的统一多模态模型。每个任务可以独立处理，参赛者可以为CT、3D TEE和手术视频帧分割任务分别训练并提交不同的模型。 ### 任务1：心脏CT中的二尖瓣分割 该任务重点关注从3D心脏CT扫描中自动进行多类解剖分割。 * **目标：** 参赛者需要分割二尖瓣环和前/后瓣叶。 * **临床意义：** 高分辨率CT分割提供了术前规划所需的精确空间几何信息，包括装置尺寸确定和经导管二尖瓣边缘修复术（TEER）或环成形术等介入治疗的修复策略选择。 ### 任务2：三维经食管超声心动图（3D TEE）中的二尖瓣分割 该任务解决从容积3D TEE（超声）数据中分割瓣膜结构的问题，这是术中瓣膜功能的关键参考。 * **目标：** 从3D超声容积中描绘瓣叶。 * **技术重点：** 算法必须足够稳健，能够处理超声特有的伪影、声学噪声、信号丢失以及瓣叶与周围心脏组织之间的模糊边界。 ### 任务3：手术视频帧中的二尖瓣分割 该任务重点关注从术中手术视频提取的单帧中提取解剖信息。 * **目标：** 从外科医生直视角度看到的二尖瓣结构的像素级分割。 * **方法论：** 虽然数据源自视频录制，但该任务被定义为基于帧的分割问题。参赛者应在训练和推理中将提供的帧视为单独的2D图像。 * **临床重点：** 该任务能够在实际介入治疗中验证瓣叶运动和结构连续性，在手术环境中提供实时反馈。

Evaluation

# 评估指标 本挑战赛中的所有任务均使用三个标准分割指标进行评估：Dice相似系数（DSC）、Hausdorff距离（HD）和平均表面距离（ASD）。这些指标衡量提交分割与参考标注之间的区域重叠和边界精度。 对于每个案例，预测结果与相应的真实标签进行比较。最终任务级结果通过对所有评估案例的案例级指标值进行平均得到。 ## 1. 指标定义 ### Dice相似系数（DSC） Dice分数衡量预测分割与真实标签之间的重叠程度。 - **范围：** [0, 1]，其中1表示完美重叠。 - **定义：** DSC计算为两个掩模的交集的两倍除以其体积或面积之和。 - **解释：** DSC值越高表示分割重叠效果越好。 ### Hausdorff距离（HD） Hausdorff距离通过测量预测结果与真实标签之间的最大表面偏差来评估边界精度。 - **定义：** HD计算为预测掩模与参考掩模之间的双向表面到表面距离的最大值。 - **实现：** 从二值掩模中提取表面，当存在图像间距元数据时使用其进行欧几里得距离计算。 - **解释：** HD值越低表示边界对齐效果越好。 ### 平均表面距离（ASD） ASD衡量预测表面与真实标签表面之间的平均距离。 - **定义：** ASD计算为从预测表面到参考表面的距离与从参考表面到预测表面的距离的对称平均值。 - **实现：** 与HD相同，距离计算在可用时使用图像间距。 - **解释：** ASD值越低表示总体形状一致性越好。 ## 2. 任务特定计算 ### 任务1：CT分割 任务1作为3D分割任务进行评估。 - **前景定义：** 标签`1` - **案例级计算：** 对每个案例的二值前景掩模计算DSC、HD和ASD。 - **最终任务结果：** 所有案例级值的平均值。 ### 任务2：TEE分割 任务2作为3D多类别分割任务进行评估。 - **前景定义：** 标签`1`和`2` - **案例级计算：** 分别为每个前景类别计算指标，并在每个案例内对类别级指标值进行平均。 - **最终任务结果：** 所有案例级值的平均值。 ### 任务3：视频/图像分割 任务3作为2D分割任务进行评估。 - **前景定义：** 预测结果被视为二值前景掩模，并与真实标签中案例特定的`target_label`进行比较。 - **案例级计算：** 对每个案例或帧计算DSC、HD和ASD。 - **最终任务结果：** 所有案例级值的平均值。 ## 3. 重要评估说明 - **间距感知型距离指标：** 当间距元数据可用时，HD和ASD使用物理间距进行计算；否则使用单位间距。 - **对称表面距离：** HD和ASD都在预测和参考表面之间双向计算。 - **缺失或无效预测：** 如果案例缺失或无法评估，将施加惩罚（`DSC = 0`，`HD = 大惩罚值`，`ASD = 大惩罚值`）。 - **空掩模处理：** 如果对于评估的类别预测和真实标签都为空，则将该案例视为完美匹配（`DSC = 1`，`HD = 0`，`ASD = 0`）。 ## 4. 排名策略 官方评估程序包分别报告每个任务的三个指标： - `Task1_DSC`、`Task1_HD`、`Task1_ASD` - `Task2_DSC`、`Task2_HD`、`Task2_ASD` - `Task3_DSC`、`Task3_HD`、`Task3_ASD` 当前评估程序包内不计算其他额外的归一化复合分数。参与者应该

Future Plans

# MVAA 挑战赛 - 发表计划与未来规划 MVAA 挑战赛具有完整的发表框架，重点关注临床相关性、方法学创新性和长期多模态基准开发。 ## 1. 会议论文集发表 * **Springer LNCS:** 我们将协调会后论文集，使参赛团队能够在与第一届 MICCAI 医学世界模型研讨会 (MWM) 相关的 **Springer 计算机科学讲义 (LNCS)** 卷中发表详细的方法描述。 * **开放获取:** 该机制确保所有成功的技术贡献在 MICCAI 生态系统中公开可用且可引用。 ## 2. 期刊特刊投稿 * **优秀投稿:** 高影响力的投稿将被邀请向 **IEEE 生物医学工程汇刊 (TBME)** 特刊提交扩展稿件，特刊题目为：*"数字孪生、基于仿真的人工智能与医学世界模型。"* * **遴选标准:** 评估超越排行榜排名，重点关注方法学创新性、跨模态泛化能力和二尖瓣分析的实际临床应用潜力。 ## 3. 挑战赛总结论文 * **期刊级总结:** 我们将协调编写综合性挑战赛总结论文，报告和分析 MVAA 结果。 * **分析内容:** 包括所有参赛方法的荟萃分析、多模态分割的主要发现，以及结构性心脏病分析的研究趋势。 * **目的:** 该论文作为社区的权威参考资料，为统一解剖学建模的未来研究方向提供指导。 ## 4. 长期未来规划 1. **年度重复举办:** 将挑战赛作为 **年度活动** 维持运营，与 MICCAI 固定截止日期相关联，持续扩展多模态临床数据集。 2. **与世界模型集成:** 进一步将基准与 **医学世界模型和数字孪生框架** 集成，将二尖瓣分析扩展至纵向和预测性手术建模。 3. **临床转化:** 推动表现最佳的算法转化为二尖瓣修复、经导管二尖瓣边缘对边缘修复 (TEER) 和瓣环成形术的临床工具，减少操作者依赖性，提高干预安全性。 4. **全球合作:** 建立多模态心脏研究可持续联盟，促进临床医生、超声医学工作者和人工智能研究人员之间的持续协作。
