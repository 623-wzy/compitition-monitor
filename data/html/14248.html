Description

# 🩺 # **OncoSegment：乳腺肿瘤分割挑战赛** # 💡 ## **为什么这个问题很有趣** 乳腺癌仍然是全球女性主要死亡原因之一。通过超声波进行早期准确检测至关重要，但解读这些图像出了名地困难。超声图像受到"斑点噪声"和低对比度的影响。就像等离子体物理模拟中的"斑点"一样，超声中的肿瘤间歇性出现、边界不规则，容易与健康组织混淆。本挑战要求你开发一个能够精确勾勒肿瘤边界的人工智能，这项任务可以直接协助放射科医生进行活检规划和治疗。 # 📊 ## **数据集** 该数据集来自乳腺超声图像（BUSI）集合，包括25-75岁女性的数据。它由标准分辨率（约500x500像素）的高质量PNG扫描图像组成。 数据集分类： 良性肿瘤：通常为卵圆形或圆形，边界光滑的肿瘤。 恶性肿瘤：癌性肿瘤，通常以"尖刺"（分叶状）或不规则边界为特征，会侵入周围组织。 正常扫描：无肿瘤的扫描（用于测试模型避免假阳性的能力）。 启动工具包中包含的内容： train_images：显示乳腺组织的PIL图像列表（灰度图）。 train_masks：二值掩膜，其中1代表肿瘤，0代表背景。 train_labels：识别肿瘤为良性或恶性的元数据。 test_images：需要预测掩膜的图像。 # 📏 ## **评估指标：测量精度** 在医学成像中，"框"是不够的。我们需要准确了解哪些像素是癌变的。我们使用两个互补的重叠指标： 1. 主要指标：Dice系数（F1分数） Dice系数是医学分割的黄金标准。它是重叠区域面积的两倍除以预测和真实标签中的像素总数。 $$Dice = \frac{2 \times |P \cap G|}{|P| + |G|}$$ 原因？它对掩膜内部的"完整性"高度敏感，奖励那些形状完全正确的模型。 2. 次要指标：Jaccard指数（IoU） Jaccard指数是交集面积除以并集面积。 $$Jaccard = \frac{|P \cap G|}{|P \cup G|}$$ 原因？ 它比Dice对小错误的"惩罚"更严厉。 它通过突出像素误差最少的模型来帮助打破排行榜并列。

# 种子 启动器 `submission.py` 框架： ```python import cv2 import numpy as np class Model: def fit(self, train_images, train_masks, train_labels=None): return self def predict(self, test_images, sample_ids=None): preds = [] for image in test_images: img = np.array(image.convert("L")) blurred = cv2.GaussianBlur(img, (5, 5), 0) _, mask = cv2.threshold( blurred, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU ) preds.append(mask) return preds def get_model(): return Model() ```
