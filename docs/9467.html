FAME'26 Grand Challenge

ID: 9467 创建者: mmosc 创建时间: 2025-07-04 参与人数: 55 提交次数: 450 在 Codabench 查看 ↗

竞赛描述

多语言环境下的面孔-声音关联（FAME）挑战赛 2026 致力于探索在多语言情景的独特条件下的面孔-声音关联。这一条件的灵感来自于全球人口中有一半是双语使用者这一事实，而且人们通常在多语言情景下进行交流。该挑战赛采用了一个名为多语言视听（MAV-Celeb）的数据集来探索多语言环境中的面孔-声音关联。

Overview

# 挑战目标 FAME 挑战赛2026计划的主要目标是为学术和工业研究人员提供一个共同平台，以开发和探索语言在人脸-声音关联中的影响，这对各种下游任务都很有用。在本挑战赛下开展的研究有望从真实场景的角度推进人脸-声音关联任务的发展。该挑战赛重点探索以下内容，但不限于： - 研究语言信息在人脸-声音关联中的影响 - 探索人脸-声音关联中特定于语言的知识 - 探索与语言无关的人脸-声音关联模型 - 与人脸-声音关联中语言不匹配相关的任何其他信息 # 任务 （左）标准人脸-声音关联通过跨模态验证任务建立。（右）FAME挑战赛2026将验证任务扩展到分析多种语言的影响。 # 挑战赛设置 MAV-Celeb数据集分为训练集和测试集，包含来自同一语言的不相交身份，通常称为未见-未闻配置。 在测试时，网络在听过和完全未听过的语言上进行跨模态验证任务的评估。数据集划分V1-EU、V2-EH分别包含64-6、78-6个身份用于训练和测试。V2-EH划分将用于进展阶段。最终评估将在V1-EU的测试集上进行。V1和V2中测试文件的每一行格式如下： ``` ysuvkz41 voices/English/00000.wav faces/English/00000.jpg tog3zj45 voices/English/00001.wav faces/English/00001.jpg ky5xfj1d voices/English/00002.wav faces/English/00002.jpg yx4nfa35 voices/English/01062.wav faces/English/01062.jpg bowsaf5e voices/English/01063.wav faces/English/01063.jpg ``` MAV-Celeb数据集是公开可用的。我们提供了来自代表人脸和声音的最先进方法的特征。 # 基准方法 基准方法采用双流管道来获取人脸和声音输入的各自嵌入。第一条流对应于在人脸模态上预训练的卷积神经网络（CNN）。我们将此CNN的倒数第二层的输出作为输入人脸图像的特征嵌入。同样，第二条流是一个预训练的音频编码网络，为输入音频信号（通常是短时谱图）输出特征嵌入。基准方法利用两种模态嵌入的互补线索形成丰富的融合嵌入，并对其施加正交约束以学习判别性的联合人脸-声音嵌入。更多信息可在我们的先前工作和GitHub存储库中获得。 # 评估指标 我们考虑使用等错误率（**EER**）作为评估挑战赛性能的指标。我们期望挑战赛参与者为每个测试对提交输出分数文件，以表示系统对人脸和声音匹配的置信度，或换句话说，人脸和声音属于同一个人的置信度。 # 结果提交 在目录中...
