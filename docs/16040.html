## MICCAI STSR 2026 预赛2D图像分割 该任务是STS26的前置任务。为防止数据滥用，完成后，你将有资格获得STS26主竞赛数据集的访问权限。请将模型在'data/test/imgs/'数据上的推理结果('data/test/masks.zip')提交到平台作为证明。(仅需约1小时) 如有任何问题，请直接联系：SemiTeethSegChallenge@outlook.com ## 📖 项目概述 👌一个使用[U-Net](https://arxiv.org/abs/1505)的简单2D图像分割任务。 <font size=4> - **步骤1：下载预赛数据和训练模型**<br> 为了防止我们数据的滥用，你可以先完成这个简单(非常简单)的预赛任务，并首先填写《数据访问协议》来获得数据访问权限。 数据网站：<br> -[Huggingface](https://huggingface.co/datasets/Ricoooo/MICCAI-STS26-Challenge-Pre-Task)<br> -[Modelscope](https://modelscope.cn/datasets/lizhii/MICCAI-STS26-Challenge-Pre-Task/summary)<br> -[百度网盘(提取码 = bm2u)](https://pan.baidu.com/s/1U090bZnMGEJQaD3jwqaQuA)<br> -[谷歌云盘](https://drive.google.com/drive/folders/1lER9eIavr99g28aTO0kuxIcos_k9FBSx?usp=sharing)<br> 或者你可以使用[STS-Skill](https://github.com/ricoleehduu/sts-skill)，就像你使用Claude Code一样，例如：<br> "使用STS-Skill https://github.com/ricoleehduu/sts-skill 帮我完成预赛。" - **步骤2：** 向在线提交系统提交结果。 - **步骤3：** 填写注册表并将协议提交到WPS表单或谷歌表单：<br> - [国内参赛者(WPS表单)](https://f.wps.cn/g/VV1UmM0O/)<br> - [国际参赛者(谷歌表单)](https://docs.google.com/forms/d/e/1FAIpQLSf4hxOwxQdkIHkSICzlD2xnlAuPWr3EvUeB7oeuRKRPUimAKQ/viewform?usp=publish-editor) </font> ## 💻 数据结构 ``` data/ ├── imgs/ # 训练图像 ├── masks/ # 训练掩码 └── test/ ├── imgs/ # 测试图像(50个样本) └── masks/ # 放置你的预测结果的位置(50个样本) ``` ## ☑️ 快速开始 代码可在[U-Net](https://arxiv.org/abs/1505)获取 ### 环境要求 - Python 3.6+ - PyTorch 1.13+ - CUDA(推荐) ### 安装依赖 ```bash pip install -r requirements.txt ``` ### 训练模型 ```bash python train.py --amp ``` ## 🛠️ 使用方法 ### 训练参数 ```console > python train.py -h usage: train.py [-h] [--epochs E] [--batch-size B] [--learning-rate LR] [--load LOAD] [--scale SCALE] [--validation VAL] [--amp] 在图像和目标掩码上训练UNet 可选参数： -h, --help 显示此帮助信息并退出 --epochs E, -e E 训练轮数 --batch-size B, -b B 批处理大小 --learning-rate LR, -l LR 学习率 --load LOAD, -f LOAD 从.pth文件加载模型 --scale SCALE, -s SCALE 图像的缩放因子 --validation VAL, -v VAL 用作验证的数据百分比(0-100) --amp 使用混合精度 ``` ### 预测 #### 批处理 ```bash python predict.py -m checkpoints/checkpoint_epoch1.pth -i data/test/imgs python predict.py -m checkpoints/checkpoint_epoch1.pth -i data/test/imgs -o data/test/masks python predict.py -m checkpoints/checkpoint_epoch1.pth -i data/test/imgs -o data/test/masks -t 0.3 ``` #### 单个文件 ```bash python predict.py -m checkpoints/checkpoint_epoch1.pth -i input.png -o output.png ``` #### 预测参数 ```console > python predict.py -h usage: predict.py [-h] --model FILE --input INPUT [INPUT ...] [--output INPUT [INPUT ...]] [--viz] [--no-save] [--mask-threshold MASK_THRESHOLD] [--scale SCALE] [--bilinear] [--classes CLASSES] 从输入图像预测掩码 可选参数： -h, --help 显示此帮助信息并退出 --model FILE, -m FILE 模型检查点
