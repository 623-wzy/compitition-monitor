Overview

# NeuroMM-2026 挑战赛 — 赛道2：NMM-Context-IED 欢迎来到 **NeuroMM-2026 多模态神经生理信息智能挑战赛的赛道2（NMM-Context-IED）**。 ### NMM-Context-IED - 主要评估指标：AUPRC - 重点：通过同步视觉上下文特征进行伪影抑制 基于视觉增强的检测与同步上下文特征。 ## 任务 二分类：对于每个样本，预测其包含癫痫**尖波**（正样本，标签 `1`）与**非尖波**（负样本，标签 `0`）的概率。赛道2 **鼓励团队将预先提取的同步视觉上下文特征与生理信号（EEG/EMG/ECG）相结合**，以抑制伪影并提高相对于仅用信号检测的鲁棒性。 ## 数据 从以下地址申请访问和下载数据集： > <https://huggingface.co/datasets/NeuroMM/NeuroMM-2026> 数据集包含**训练集**、**验证集**和**候选集**（包含赛道2的测试样本与干扰项混合）。候选集故意大于官方赛道2集合，**参赛者不被告知哪些样本会被计分** — 对发布的候选集中的*所有*id进行预测并提交所有结果；评分系统将自动只保留官方隐藏的赛道2样本并忽略其余样本。 ## 注册规则 - 一个人只能参加**一个团队**。 - 一个团队可参加多个赛道，但**每个赛道只有一名团队成员账号**可进行提交。 - 在 **Codabench 上注册所用的电子邮箱必须与** HuggingFace 数据集访问申请表中填写的电子邮箱**相匹配**；否则注册**将不被批准**。所有注册都由组织者进行人工审核。 ## 提交 本竞赛仅接受**预测结果**。您上传单个预测文件（打包为 ZIP 格式） — 不需要提交代码或模型。具体格式见**评估**页面。 ## 排名 提交结果按 **AUPRC**（精准率-召回率曲线下面积）排名。这是**唯一的**评估指标 — 不计算、报告或显示其他指标。

Evaluation

# 评估 ## 指标 **AUPRC** — 精确率-召回率曲线下的面积（又称平均精确率），以 `label = 1`（尖峰）作为正类计算。这是**唯一**的指标；不生成、报告或显示其他指标。数值越高越好。 ## 提交格式 提交一个**ZIP**文件，其中包含恰好一个名为**`submission.csv`**的文件： ``` sample_submission.zip └── submission.csv ``` > 仅接受文件名 `submission.csv`。任何其他名称（例如 `prediction.csv`）都会导致提交失败。 `submission.csv` 包含两列： ```csv sample_id,prediction NMM26-XXXXXXXX,0.0123 NMM26-YYYYYYYY,0.9871 NMM26-ZZZZZZZZ,0.5040 ``` - `sample_id` — 与发布的候选集中提供的id完全相同。 - `prediction` — 一个实数值，表示**样本为尖峰的概率/分数**（正类）。任何单调分数都适用于AUPRC；不要求值经过校准（推荐使用 `[0, 1]` 范围）。 ### 重要事项 - 对已发布候选集中的**每个**id进行预测，并包含所有这些id。**不会**告知哪些id是官方Track-2样本。 - 评分器仅保留隐藏的官方Track-2 id，**默认忽略所有其他行**（干扰项/其他赛道）。额外的行没有问题。 - 如果以下情况发生，提交**失败**：任何官方Track-2样本缺少预测、预测不是数值、`sample_id` 重复，或文件名不是 `submission.csv`。

Terms

# 条款和条件 参与NeuroMM-2026挑战赛（赛道2：NMM-Context-IED）即表示您同意： - **仅**将提供的数据用于研究和参加本挑战赛。禁止数据的再分发或任何商业/未授权使用。 - 使用自动化机器学习方法进行预测。通过人工检查生理信号对样本进行手动标注**不**是可接受的方法，违反了挑战赛的精神。 - 不尝试对样本进行去匿名处理、识别哪些候选ID是评分的官方样本，或以其他方式对隐藏标签进行逆向工程。 - 进行合理数量的提交（执行每日提交上限）。禁止滥用评估服务来探测隐藏标签。 组织者可在必要时更新这些规则和竞赛日程，并保留对违反这些条款的提交进行取消资格处理的权利。 数据集在NeuroMM-2026数据使用政策下提供（仅限学术研究和挑战赛参与）。
