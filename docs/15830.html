Overview

# MEG脑编码挑战 ## 目标 构建一个模型，预测人类大脑对自然自由观看视觉场景的反应。具体地说，您将预测**注视对齐的MEG活动**——在每次眼动注视开始后110毫秒时在204个MEG梯度计通道上测得的神经反应模式。 ## 背景 数据来自**主动视觉语义学（AVS）**数据集，其中参与者在同时记录眼动和全脑MEG的情况下自由探索自然场景照片。 您获得来自五名训练受试者（受试者1-5）的数据，并被要求推广到**保留的受试者（受试者60）**，其场景观看行为和相关的MEG反应构成评估集。 ## 任务 **挑战1——110毫秒处的快照编码** 给定注视元数据（场景ID、凝视坐标、持续时间、序列位置等）和对应的场景图像，预测注视开始后110毫秒时204个MEG梯度计振幅的向量。 **提交形状：** `(7750, 204)` ——开发集中每次注视一行，204个通道。 ## 时间表 | 日期 | 事件 | |------|------| | 2026年4月 | 挑战开放 | | 2026年7月 | 挑战关闭 | ## 限制 - 每天5次提交 - 总共50次提交

Evaluation

# 评估 ## 指标 提交的作品使用**皮尔逊相关系数（r）**进行评估，该系数在开发集的所有7750个注视点中**独立计算每个204个通道**的预测MEG振幅与实际MEG振幅之间的相关性。 这产生一个向量 **r** ∈ ℝ^204，从中导出四个排行榜分数： | 列 | 定义 | |--------|-----------| | **平均 r** *（主要指标）* | `mean(r)` — 所有通道的平均编码 | | **中位数 r** | `median(r)` — 稳健的中心趋势 | | **最大 r** | `max(r)` — 最佳单通道编码 | | **最差10个通道** | `mean(r[argsort(r)[:10]])` — 最差10个通道的平均值 | 排名由**平均 r** 决定（越高越好）。所有其他列仅用于诊断目的。 ## 提交格式 1. 创建一个形状为 `(7750, 204)` 且数据类型为 `float64` 的NumPy数组。 - 行顺序必须与 `subject60/challenge1_dev/metadata.csv` 完全匹配。 - 列顺序必须与 `channel_names.txt` 完全匹配（MEG0001 … MEG0204）。 2. 将其保存为 `predictions.npy`： ```python import numpy as np np.save("predictions.npy", pred) # 形状 (7750, 204) ``` 3. 压缩它（文件必须在zip的根目录下，不能在子文件夹中）： ```bash zip submission.zip predictions.npy ``` 4. 通过提交按钮上传 `submission.zip`。 ## 有效性检查 评分程序会拒绝以下提交： - 形状错误（除7750 × 204外的任何形状） - 包含NaN或Inf值 无效提交在所有列中得分为0.0。
