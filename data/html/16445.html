Overview

# NeuroMM-2026 挑战赛 — 赛道 3: NMM-Source-IED 欢迎参加 NeuroMM-2026 多模态神经生理智能挑战赛的 **赛道 3 (NMM-Source-IED)**。 ### NMM-Source-IED - 主要指标：加权-F1 - 重点：癫痫灶区域的 5 分类定位 结合视觉上下文特征和生理信号来定位癫痫焦点。 ## 任务 **5 分类**：对于每个样本，预测癫痫灶区域类别（整数范围在 **{1, 2, 3, 4, 5}**）。赛道 3 **鼓励参赛队伍将预提取的同步视觉上下文特征与生理信号（EEG/EMG/ECG）相结合**来定位癫痫焦点。 ## 数据 申请访问权限并从以下网址下载数据集： > <https://huggingface.co/datasets/NeuroMM/NeuroMM-2026> 数据集包含**训练集**、**验证集**和**候选集**（候选集包含赛道 3 测试样本与干扰项的混合）。候选集故意设置得大于官方赛道 3 集合，**您不会被告知哪些样本被计分** — 请对发布的候选集中的*每个* id 进行预测并提交全部结果；评分系统会自动仅保留官方隐藏的赛道 3 样本并忽略其余样本。 ## 注册规则 - 一个人只能参加**一个团队**。 - 一个团队可以参加多个赛道，但**每个赛道仅一名团队成员账户**可以提交。 - 在 **Codabench 上注册的电子邮件必须与**填写的 HuggingFace 数据集访问申请表中的电子邮件一致；否则注册**将不被批准**。所有注册均由组织者手动审核。 ## 提交 本竞赛仅接受**预测结果**。您上传单个预测文件（打包为 ZIP 格式）— 无需提交代码或模型。有关确切格式，请参阅**评估**页面。 ## 排名 提交按 **加权-F1**（5 个类别的各类 F1 的支持加权平均）排名。这是**唯一**的指标 — 不计算、报告或显示其他指标。

Evaluation

# 评估 ## 指标 **加权F1** — 5个类别的逐类F1的支持度加权平均值（`scikit-learn` `f1_score(average='weighted')`）。这是**唯一**的指标；不产生、报告或显示任何其他指标。数值越高越好。 ## 提交格式 提交一个**ZIP**文件，其中恰好包含一个名为**`submission.csv`**的文件： ``` sample_submission.zip └── submission.csv ``` > 仅接受文件名`submission.csv`。任何其他名称（例如`prediction.csv`）都会导致提交失败。 `submission.csv`包含两列： ```csv sample_id,prediction NMM26-XXXXXXXX,3 NMM26-YYYYYYYY,1 NMM26-ZZZZZZZZ,5 ``` - `sample_id` — 发布的候选集中给定的样本ID。 - `prediction` — 预测的**类别，为整数{1, 2, 3, 4, 5}**（痫源区类别）。每个样本一个类别。 ### 重要事项 - 为发布的候选集中的**每个**ID进行预测，并将其全部包含在内。**不会告知**您哪些ID是官方Track-3样本。 - 评分器仅保留隐藏的官方Track-3 ID，**自动忽略所有其他行**（干扰项/其他赛道）。额外的行没有问题。 - 如果任何官方Track-3样本没有预测、预测非数字、`sample_id`重复，或文件未命名为`submission.csv`，提交将**失败**。

Terms

# 条款和条件 参与NeuroMM-2026挑战赛（赛道3：NMM-Source-IED）即表示您同意： - **仅**将提供的数据用于研究和本挑战赛的参与。禁止数据的再分发、任何商业使用或未授权使用。 - 采用自动化机器学习方法进行预测。通过人工检查生理信号对样本进行手动标注**不是**可接受的方法，违反了挑战赛的宗旨。 - 不尝试对样本进行去匿名化处理、识别哪些候选ID是评分的官方样本，或以其他方式对隐藏标签进行逆向工程。 - 进行合理数量的提交（执行每日限制）。禁止滥用评估服务来探测隐藏标签。 组织者可在必要时更新这些规则和竞赛日程，并保留对违反这些条款的提交进行取消资格的权利。 该数据集根据NeuroMM-2026数据使用政策提供（仅用于学术研究和挑战赛参与）。
