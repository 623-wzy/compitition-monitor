Overview

# NeuroMM-2026 挑战赛 — 赛道 3：NMM-Source-IED 后评估 欢迎参加 **NeuroMM-2026 多模态神经生理智能挑战赛** **赛道 3（NMM-Source-IED）**。 竞争阶段已结束。该平台在有限时间内开放，严格用于消融研究；这些后续结果将不纳入官方排行榜排名。 ### NMM-Source-IED - 主要指标：加权 F1 值 - 重点：癫痫病灶区域的 5 分类定位 结合视觉上下文特征与生理信号来定位癫痫灶。 ## 任务 **5 分类**：对于每个样本，预测癫痫病灶区域类别（整数，范围为 **{1, 2, 3, 4, 5}**）。赛道 3 **鼓励团队结合预提取的同步视觉上下文特征与生理信号（脑电图/肌电图/心电图）** 来定位癫痫灶。 ## 数据 从以下网址申请访问权限并下载数据集： > <https://huggingface.co/datasets/NeuroMM/NeuroMM-2026> 该数据集包含 **训练集**、**验证集** 和 **候选集**（包含混合了干扰项的赛道 3 测试样本）。候选集故意设置得比官方赛道 3 集合更大，**不会告知您哪些样本会被计分** — 请对发布的候选集中的所有 id 进行预测并提交所有结果；计分器将自动仅保留官方隐藏的赛道 3 样本，忽略其余部分。 ## 注册规则 - 一个人只能参加 **一个团队**。 - 一个团队可参加多个赛道，但对于 **每个赛道，只有一个团队成员的账户** 可进行提交。 - 在 **Codabench 上注册的电子邮件必须与** HuggingFace 数据集访问请求表单中填写的电子邮件 **匹配**；否则注册将 **不被批准**。所有注册由组织者手动审核。 ## 提交 本竞赛仅接受 **预测结果**。您上传单个预测文件（打包为 ZIP 格式）— 不需要提交代码或模型。详见 **评估** 页面了解确切格式。 ## 排名 提交成果按 **加权 F1 值**（5 个类别的按支持度加权平均的每类 F1）排名。这是 **唯一的** 指标 — 不计算、报告或显示其他指标。

Evaluation

# 评估 ## 指标 **加权F1** — 5个类别的每类F1的支持度加权平均值（`scikit-learn` `f1_score(average='weighted')`）。这是**唯一的**指标；不产生、报告或显示其他指标。数值越高越好。 ## 提交格式 提交一个**ZIP**文件，其中恰好包含一个名为**`submission.csv`**的文件： ``` sample_submission.zip └── submission.csv ``` > 仅接受文件名`submission.csv`。任何其他名称（例如`prediction.csv`）都将导致提交失败。 `submission.csv`有两列： ```csv sample_id,prediction NMM26-XXXXXXXX,3 NMM26-YYYYYYYY,1 NMM26-ZZZZZZZZ,5 ``` - `sample_id` — 与发布的候选集中给定的id完全相同。 - `prediction` — 预测的**类别为整数{1, 2, 3, 4, 5}**（癫痫灶区域类别）。每个样本一个类别。 ### 重要事项 - 为发布的候选集中的**每个**id进行预测并包括所有id。**不会**告知您哪些id是官方Track-3样本。 - 评分器仅保留隐藏的官方Track-3 id，**以静默方式忽略所有其他行**（干扰项/其他赛道）。额外行没有问题。 - 如果任何官方Track-3样本没有预测、预测不是数字、`sample_id`重复或文件未命名为`submission.csv`，提交**失败**。

Terms

# 条款和条件 参与NeuroMM-2026挑战赛（赛道3：NMM-Source-IED）即表示您同意： - 提供的数据**仅**用于研究和参加本挑战赛。禁止数据的重新分发或任何商业/未授权使用。 - 使用自动化机器学习方法进行预测。通过人工检查生理信号对样本进行手动标注**不**是可接受的方法，违反了挑战赛的精神。 - 不尝试对样本进行匿名化识别、识别哪些候选者ID是评分的官方样本，或以其他方式对隐藏标签进行逆向工程。 - 进行合理数量的提交（实施每日上限）。禁止滥用评估服务以探测隐藏标签。 组织者可在必要时更新这些规则和比赛时间表，并保留取消违反这些条款的提交资格的权利。 该数据集根据NeuroMM-2026数据使用政策提供（仅限学术研究和挑战赛参与）。
