# 🏆 MISP Challenge 2026 音频-视觉-文本多模态关键词识别挑战 ## 📋 概述 本竞赛专注于现实语音处理场景中的音频-视觉-文本多模态关键词识别任务。 ### 任务描述 您的模型需要判断给定的音频-视频对是否来自同一说话者和同一关键词。 ### 内集合 vs 外集合 | 术语 | 描述 | |------|------| | **inset（内集合）** | 训练过程中看到的关键词 | | **outset（外集合）** | 训练过程中未见过的新关键词 | ### 评估指标 - **AUC_INSET** 内集合关键词上的AUC 越高越好 - **EER_INSET** 内集合上的EER (%) 越低越好 - **AUC_OUTSET** 外集合关键词上的AUC 越高越好 - **EER_OUTSET** 外集合上的EER (%) 越低越好 ## 🚀 快速开始 - 提交的5个步骤 1. **生成预测** 在测试集上运行您的模型并收集预测结果 2. **准备CSV** 以所需的CSV格式保存预测结果 3. **压缩CSV**
