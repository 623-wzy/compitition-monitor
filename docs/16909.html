# 参赛指南 ArGuard 2026 — 任务1 / Track A · **子任务A1：仇恨表情包二分类检测** ## 参赛资格 您可以以个人或团队身份参加。一个人可以参加ArGuard 2026内的多个共享任务——详见[任务网站](https://araieval.gitlab.io/ArGuard2026/)的完整列表。 ## 如何注册 1. 在本CodaBench竞赛页面上注册。 2. 阅读[条款](#)选项卡——特别注意提交文件格式和命名规范。 3. 从Hugging Face Hub下载数据：[`QCRI/ArGuard-Task1`](https://huggingface.co/datasets/QCRI/ArGuard-Task1)。 4. 克隆启动工具包：[`araieval/ArGuard-2026-tasks`](https://github.com/araieval/ArGuard-2026-tasks)。 5. 在开发阶段对`dev_test`分割集训练模型并生成预测，在最终评估阶段对`test`分割集进行预测。 ## 提交 - 每个阶段由一名团队成员通过**提交**选项卡提交。 - 将TSV预测文件上传为`prediction.zip`（见条款）。 - 截止日期前最后一次有效提交将作为该阶段团队的最终提交。 ## 系统描述论文 鼓励所有团队为研讨会提交系统描述论文。如果您参加了多个子任务，单篇论文可涵盖多个子任务。 ## 联系方式 问题咨询：arguard2026-organizers@googlegroups.com

Dataset

# 数据集 ArGuard Task 1 — Track A 基于阿拉伯语多模态表情包数据集构建：每个实例包含一张图片和叠加在其上的阿拉伯文本（通过OCR提取）。表情包已手动标注仇恨属性。 对于**子任务A1**，每个表情包都有一个来自{`Hateful`（仇恨）、`Not Hateful`（非仇恨）}的单一二元标签。 ## 获取数据 官方数据集位于 Hugging Face Hub 的[`QCRI/ArGuard-Task1`](https://huggingface.co/datasets/QCRI/ArGuard-Task1)。使用启动工具包中的辅助脚本下载： ```bash git clone https://github.com/araieval/ArGuard-2026-tasks.git cd ArGuard-2026-tasks/task1 pip install -r requirements.txt python data/download_data.py ``` ## 数据分割 | 数据集 | 记录数 | 标签 | 阶段 | 用途 | |--------------|--------:|----------|--------------------------|--------------------------------------------| | `train` | 3,500 | 完整 | 开发阶段 | 模型训练 | | `dev` | 500 | 完整 | 开发阶段 | 超参数选择、本地评估 | | `dev_test` | 500 | **无** | 开发阶段（排行榜）| 开发阶段排行榜的提交目标 | | `test` | 500 | 完整 | 最终评估 | 盲测排名（三重标注黄金标准） | `dev_test` 数据集以标签被移除的形式发布——这是开发阶段提交预测结果的文件。 ## 记录架构（运行 `download_data.py` 后） ```json { "id": "f9a8…b1.jpg", "image_path": "img/f9a8…b1.jpg", "text": "OCR提取的阿拉伯文本…", "label": "Hateful" | "Not Hateful" | null, "fine_grained_label": ["Mocking", "Incitement"], "annotations": [] } ``` ## 许可证 数据集采用 **CC BY-NC 4.0** 许可证发布（仅供非商业研究使用）。 ## 资源 - 启动工具包：<https://github.com/araieval/ArGuard-2026-tasks> - 任务网站：<https://araieval.gitlab.io/ArGuard2026/>
