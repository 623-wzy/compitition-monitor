# 如何参与 ArGuard 2026 — 任务1 / A赛道 · **子任务A1：仇恨表情包二分类检测** ## 谁可以参加 您可以以个人或团队身份参加。一个人可以参加ArGuard 2026内的多个共享任务 — 查看[任务网站](https://araieval.gitlab.io/ArGuard2026/)获取完整列表。 ## 如何注册 1. 在此CodaBench竞赛页面上注册。 2. 阅读[条款](#)标签 — 特别注意提交文件格式和命名规范。 3. 从Hugging Face Hub下载数据：[`QCRI/ArGuard-Task1`](https://huggingface.co/datasets/QCRI/ArGuard-Task1)。 4. 克隆初始工具包：[`araieval/ArGuard-2026-tasks`](https://github.com/araieval/ArGuard-2026-tasks)。 5. 训练您的模型，在开发阶段生成`dev_test`分割上的预测，在最终评估阶段生成`test`分割上的预测。 ## 提交 - 每个阶段由一名团队成员通过**提交**标签进行提交。 - 将您的TSV预测文件作为`prediction.zip`上传（见条款）。 - 截止时间前的最新有效提交将被视为您团队在该阶段的最终提交。 ## 系统描述论文 所有团队都可以为研讨会提交系统描述论文。如果您参加了多个子任务，一篇论文可以涵盖多个子任务。 ## 联系方式 问题咨询：arguard2026-organizers@googlegroups.com

Dataset

# 数据集 ArGuard 第1任务 — A赛道基于阿拉伯语多模态表情包数据集构建：每个实例包含一张图像和覆盖在其上的阿拉伯文本（通过OCR提取）。表情包已手动标注仇恨程度。 对于**子任务A1**，每个表情包都有一个来自{`Hateful`（仇恨）、`Not Hateful`（非仇恨）}的二分类标签。 ## 获取数据 官方数据集存放在Hugging Face Hub的[`QCRI/ArGuard-Task1`](https://huggingface.co/datasets/QCRI/ArGuard-Task1)。使用初始工具包中的辅助脚本下载： ```bash git clone https://github.com/araieval/ArGuard-2026-tasks.git cd ArGuard-2026-tasks/task1 pip install -r requirements.txt python data/download_data.py ``` ## 数据划分 | 划分 | 记录数 | 标签 | 阶段 | 用途 | |--------------|--------:|----------|--------------------------|--------------------------------------------| | `train` | 3,500 | 完整 | 开发阶段 | 模型训练 | | `dev` | 500 | 完整 | 开发阶段 | 超参数选择、本地评估 | | `dev_test` | 500 | **无** | 开发阶段（排行榜）| 开发阶段排行榜提交目标 | | `test` | 500 | 完整 | 最终评估 | 盲目排名（三重标注黄金标准） | `dev_test`划分发布时已移除标签——这是开发阶段进行预测提交的文件。 ## 记录模式（`download_data.py`之后） ```json { "id": "f9a8…b1.jpg", "image_path": "img/f9a8…b1.jpg", "text": "OCR提取的阿拉伯文本…", "label": "Hateful" | "Not Hateful" | null, "fine_grained_label": ["Mocking", "Incitement"], "annotations": [] } ``` ## 许可证 数据集发布于**CC BY-NC 4.0**许可证（仅限非商业研究使用）。 ## 资源 - 初始工具包：<https://github.com/araieval/ArGuard-2026-tasks> - 任务网站：<https://araieval.gitlab.io/ArGuard2026/>
