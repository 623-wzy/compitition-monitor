Overview

# ECCV 2026视觉事件多模态理解挑战赛：空间推理赛道 欢迎参加多模态大语言模型(MLLMs)事件理解挑战赛，官方主办方为ECCV 2026！ ## 🎯 概述与目标 本挑战赛基准测试MLLMs在事件型视觉上的空间推理能力，重点关注物体计数、绝对距离估计和空间关系推理。 ## 📝 任务描述 为全面评估模型能力，本挑战赛在3个不同的空间任务上评估性能： * **物体计数**：准确枚举场景中存在的特定动态或静态物体的数量。 * **绝对距离**：估计物体之间或从相机到主体的物理定量距离（例如，以米为单位）。 * **空间关系**：确定多个物体的相对拓扑位置和结构方向。 ## 📅 重要日期 *所有截止时间均为协调世界时(UTC)晚上11:59。* * **2026年5月10日**：挑战赛官网开放。 * **2026年5月12日**：训练与验证数据发布。 * **2026年5月20日**：验证服务器上线。 * **2026年6月25日**：测试数据发布及测试服务器上线。 * **2026年7月3日**：测试提交截止及测试服务器关闭。 * **2026年7月10日**：结果公布*(暂定)*。 * **待定**：技术报告提交。 ## 📊 数据统计(测试集) 该基准测试包含严格标注的测试集，涵盖以下分布： | 任务类别 | 项目数量 | | :--- | :--- | | **绝对距离** | 1202 | | **物体计数** | 414 | | **空间关系** | 301 | ## 🔗 提供的资源 为帮助您快速上手并构建模型，请参考我们的官方资源： * **训练数据**：[通过ModelScope下载](https://modelscope.cn/datasets/XduSyL/EventMind/files) * **学术论文**：[在arXiv上阅读(2511.18448)](https://arxiv.org/pdf/2511.18448) * **项目主页**：[访问EventBench官网](https://xdusyl.github.io/eventbench.github.io/) ## ⚠️ 挑战赛规则 为维护公平性和科学完整性，所有参赛者必须严格遵守以下规则： 1. **禁止作弊或探测**：严格禁止任何探测测试服务器、操纵提交以提取测试集答案或使用测试数据进行模型训练的行为，违者将被立即取消参赛资格。 2. **模型规模限制**：为鼓励设计高效且易获取的模型，任何评估模型的参数总数必须**≤10B**(100亿参数)。

Evaluation Details

# 评估指标与提交指南 ## 提交格式 您的提交必须是单个JSON文件。该文件必须包含一个**平面字典**（键值对），其中键是测试问题`id`，值是您的模型预测的`answer`。 *(注意：请确保您的提交是直接的JSON对象`{...}`而非包装在JSON数组`[{...}]`中，以确保与我们的评分引擎兼容。)* **`predictions.json`示例：** ```json { "228350c5-4a8f-4976-abf9-fe52b86bddae": "D", "a959e02d-7ef6-457b-9a16-9c85934ba980": "5.3198", "e495b370-59ba-471b-8642-5ea239380f5a": "4", "d42e3a21-2c72-48ef-9f33-522338dc1c44": "left", "97258721-3d86-4bc2-82d1-481b59e1b7cd": "spatial\ntemporal" } ``` ## 如何提交 1. **准备文件：** 格式化您的模型预测结果并严格保存为`predictions.json`。 2. **压缩：** 将`predictions.json`文件直接压缩为`.zip`压缩包（例如`submission.zip`）。**不要**在压缩包中包含任何父文件夹。 3. **上传：** 导航至挑战页面的**"我的提交"**标签，选择活跃阶段，上传您的`.zip`文件。评分服务器将自动处理您的结果。 ## 指标 该挑战根据3个不同的任务对模型进行评估。评分服务器基于以下具体匹配规则计算每个类别的准确度： * **绝对距离**：直接距离值输出（例如`5.3198`）。**指标：** 预测值与真实值之间的相对误差≤30%时判定正确。 * **物体计数**：数值。物体计数的直接输出（例如`4`）。**指标：** 需要精确匹配。 * **空间关系**：字符匹配（例如`left`）。**指标：** 需要精确匹配。 ## 最终评分 评分脚本将独立计算所有3个任务的单个准确度分数。最终排行榜排名严格由**平均分数**决定，即所有任务类别分数的平均值。
