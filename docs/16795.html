Challenge Overview

# 鱼类物种识别挑战赛 ## 概述 欢迎参加鱼类物种识别挑战赛！您的任务是构建一个分类器，从图像中识别鱼类物种。 您将获得： - **训练集**：标注了物种名称的鱼类图像 - **测试集**：未标注的鱼类图像，用于开发阶段和最终阶段 从"如何参赛"页面提供的链接下载完整数据集。 ## 任务 根据鱼类图像预测其物种标签。提交内容为JSON文件，映射图像文件名到预测的物种名称。 ## 评估方法 提交内容根据**分类准确率**进行评分——测试图像中预测物种与真实标签匹配的比例。预测结果通过**图像文件名键**与真实标签匹配，因此顺序无关。 ## 数据格式 **训练标签**（`train_labels.json`）： ```json { "train/img_001.jpg": "salmon", "train/img_002.jpg": "tuna" } ``` **提交文件**（`prediction.json`）——每个测试图像一条记录： ```json { "img_001.jpg": "salmon", "img_002.jpg": "tuna", "img_003.jpg": "bass" } ``` ## 快速开始 从开发阶段页面下载入门工具包，其中包括示例提交和基础代码。

How to Participate

# 参与方式 ## 1. 下载数据集 从 Google Drive 下载鱼类图像数据集：**[TODO: 在此插入您的 Google Drive 链接]** 压缩包包含： - `train/` — 训练图像，按物种分类到子文件夹 - `train_labels.json` — 图像文件名到物种标签的映射 - `test/` — 测试图像（无标签） ## 2. 构建您的模型 在训练数据上训练鱼类物种分类器。您可以自由使用任何框架（PyTorch、TensorFlow、scikit-learn 等）和任何预训练骨干网络。 ## 3. 格式化您的预测 创建名为 `prediction.json` 的文件，将每个测试图像文件名映射到您预测的物种标签： ```json { "img_001.jpg": "salmon", "img_002.jpg": "tuna", "img_003.jpg": "bass", "img_004.jpg": "herring" } ``` 基准真值中的每个键都必须出现在您的提交中 — 额外的键将被忽略，缺少的键将导致错误。 ## 4. 提交 压缩您的预测文件并上传： ```bash zip submission.zip prediction.json ``` 然后在开发阶段页面上上传 `submission.zip`。 ## 5. 阶段 - **开发阶段**：可多次提交以调整您的模型。 - **最终评估阶段**：仅一次提交机会。请仔细选择您最好的模型。 有关截止日期，请参阅**时间表**页面。
