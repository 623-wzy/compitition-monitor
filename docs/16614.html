# 第二届低阶视觉前沿研讨会（LoViF）@ ECCV 2026 # LoViF 2026 第三届双焦点图像日夜雨滴去除挑战赛 @ ECCV 2026 ## 数据概览 数据集名称为RainDrop Clarity，由Yeying Jin等人（NUS）贡献。该数据集包含白天和夜间场景的训练和测试数据。(i) 白天雨滴数据集包含共5,442个配对或三元组图像，其中4,713对用于训练，其余729对用于测试。(ii) 夜间雨滴数据集包含9,744个配对或三元组图像，其中8,655对分配给训练集，其余1,089对用于验证和测试集。 本竞赛使用的数据集构成如下： - 训练数据：提供共14,139张日夜双焦点雨滴图像及其对应的清晰背景图像。 - 验证数据：提供共240张日夜雨滴图像。 - 测试数据：提供共731张日夜雨滴图像。 ## 数据访问 **参赛者访问数据即表示隐式同意LoViF挑战赛的条款和条件。** - **发展阶段/学习：** - 训练数据，请通过Google Drive下载[白天雨滴数据](https://drive.google.com/file/d/1lHOumI4wDsJfgGnDPOCXM6h_C1F5ZQEA/view)和[夜间雨滴数据](https://drive.google.com/file/d/1_ruwsBCzbOEkpqcqHIeKoyCg6P2Sxclr/view)。参赛者可以使用公开可用的外部数据集进行训练，但必须在信息表中明确说明数据来源。更多关于训练数据的详细信息可在FoundIR中找到。 - [验证数据](https://drive.google.com/file/d/1t2W0OC8qXZ461EIxNNFHwzVtyCgasQs2/view?usp=drive_link)（仅输入图像）已发布 - 测试数据（仅输入图像）：待定 ## 最终测试阶段提交指南（信息表可在[待定]找到） 最终提交应通过电子邮件发送给组织者（详见"组织者"选项卡中的联系方式）。 邮件主题应为：[LoViF2026] - [任务名称] - [团队名称] 邮件正文应包括以下内容： 1. 挑战赛全名 2. 团队名称 3. 团队负责人及全部成员的名字、邮箱地址和所属机构 4. 与LoViF 2026赞助商关联的团队成员 5.
