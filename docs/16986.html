# 废弃物分类检测挑战赛 ## 关于 构建最佳目标检测模型，从废弃物分类设施的真实传送带图像中检测和分类废弃物。在19个废弃物类别中竞争。 ## 数据集 - 训练集：5,238张图像，YOLO格式标签 - 测试集：1,312张图像 - 格式：YOLO（class x_center y_center width height） ## 类别（共19类） | ID | 类别 | ID | 类别 | |----|-------|----|-------| | 0 | pet-bottle-clear | 10 | sachet | | 1 | pet-bottle-green | 11 | tetrapack | | 2 | ldpe-clear | 12 | cardboard-brown | | 3 | ldpe-hm | 13 | paper-box | | 4 | ldpe-black | 14 | coconut-shell | | 5 | hdpe-bottle | 15 | footwear | | 6 | metal-can | 16 | idpe-colored | | 7 | milk-packet | 17 | hard-plastic | | 8 | pp-bag | 18 | other | | 9 | mlp-packet | | | ## 评估指标 主要指标：**mAP50**（COCO风格的平均精度均值，IoU=0.50） ## 参赛方式 1. SSH连接到分配的工作站（凭据另行共享） 2. 在提供的数据集上训练模型，数据集位置：`/home/user/hackathon/Waste_sorting_dataset/` 3. 在测试集上生成预测结果 4. 使用提供的`submit.py`脚本提交 ## 提交格式 提交包含`predictions.json`的ZIP文件，格式为COCO检测格式： ```json [ { "image_id": 0, "category_id": 0, "bbox": [x, y, width, height], "score": 0.95 } ] ``` ## 规则 - 每天最多3次官方提交 - 仅在提供的数据集上训练 - 禁止团队间共享模型权重
