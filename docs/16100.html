Overview

# 事件型多模态理解挑战赛：通用理解赛道 @ECCV 2026 欢迎来到事件相机多模态大语言模型（MLLMs）理解挑战赛，官方承办于ECCV 2026！ ## 🎯 概述与目标 本挑战赛旨在系统评估和推进多模态大语言模型在事件型视觉数据理解和推理方面的能力。与传统RGB相机不同，事件相机以异步方式捕捉亮度变化，具有高时间分辨率和高动态范围。我们的目标是促进开发鲁棒、参数高效的多模态模型，使其能够在复杂事件流中进行深层语义理解、时空推理和细粒度识别。 ## 📝 任务描述 为了全面评估模型能力，本挑战赛围绕8个核心任务进行评估： * **细节理解**：理解事件流中的细粒度视觉细节、运动和复杂交互。 * **因果推理**：推断动态事件之间的因果关系和物理逻辑。 * **动作识别**：准确识别和分类人类动作或动态活动。 * **手势识别**：从高速事件信号中识别特定手部姿态和微运动。 * **事件文字识别**：在具有挑战性的光照或快速运动场景下检测、提取和识别文本内容。 ## 📅 重要日期 *所有截止日期均为UTC时间晚上11:59。* * **2026年5月10日**：挑战赛网站开放。 * **2026年5月12日**：发布训练和验证数据。 * **2026年5月20日**：验证服务器上线。 * **2026年6月25日**：发布测试数据及测试服务器上线。 * **2026年7月3日**：测试提交截止及测试服务器关闭。 * **2026年7月10日**：结果公布 *(待定)*。 * **待定**：技术报告提交。 ## 📊 数据统计（测试集） 基准数据集包含严格标注的测试集，涵盖以下分布： | 任务类别 | 数据项数量 | | :--- | :--- | | **手势识别** | 894 | | **细节理解** | 483 | | **动作识别** | 365 | | **因果推理** | 180 | | **事件文字识别** | 162 | ## 🔗 提供的资源 为帮助您快速入门并构建模型，请参考我们的官方资源： * **训练数据**：[通过ModelScope下载](https://modelscope.cn/datasets/XduSyL/EventMind/files) * **学术论文**：[在arXiv上阅读 (2511.18448)](https://arxiv.org/pdf/2511.18448) * **项目主页**：[访问EventBench网站](https://xdusyl.github.io/eventbench.github.io/) ## ⚠️ 挑战赛规则 为维护公平性和科学完整性，所有参赛者必须严格遵守以下规则： 1. **禁止作弊或探测**：严禁任何探测测试服务器、通过操纵提交以提取测试集答案或使用测试数据进行模型训练的行为，违者将被立即取消资格。 2. **模型规模限制**：为鼓励高效和易获取的模型设计，任何评估模型的总参数数量必须 **<= 10B**（100亿参数）。

Evaluation Details

# 评估指标和提交指南 ## 提交格式 您的提交必须是一个单一的JSON文件。该文件必须包含一个**平面字典**（键值对），其中键是测试问题的`id`，值是您模型预测的`answer`。 *(注意：请确保您的提交是直接的JSON对象`{...}`而NOT被包装在JSON数组`[{...}]`中，以确保与我们的评分引擎兼容。)* **`predictions.json`示例：** ```json { "228350c5-4a8f-4976-abf9-fe52b86bddae": "D", "a959e02d-7ef6-457b-9a16-9c85934ba980": "5.3198", "e495b370-59ba-471b-8642-5ea239380f5a": "4", "d42e3a21-2c72-48ef-9f33-522338dc1c44": "left", "97258721-3d86-4bc2-82d1-481b59e1b7cd": "spatial\ntemporal" } ``` ## 如何提交 1. **准备文件：** 格式化您的模型预测并严格保存为`predictions.json`。 2. **压缩：** 直接将`predictions.json`文件压缩到`.zip`存档中（例如，`submission.zip`）。**不要**在存档中包含任何父文件夹。 3. **上传：** 导航到挑战页面上的**"我的提交"**选项卡，选择活跃阶段，并上传您的`.zip`文件。评估服务器将自动处理您的结果。 ## 指标 该挑战跨5个不同的任务评估模型。评估服务器根据以下具体匹配规则计算每个类别的准确度： * **详细理解**：单选题。**指标：** 精确匹配（A、B、C或D）。 * **动作识别**：单选题。**指标：** 精确匹配（A、B、C或D）。 * **手势识别**：单选题。**指标：** 精确匹配（A、B、C或D）。 * **因果推理**：单选题。**指标：** 精确匹配（A、B、C或D）。 * **事件OCR**：字符匹配。**指标：** 需要精确匹配。*(注意：请使用换行符`\n`表示换行，例如，`spatial\ntemporal`)*。 ## 最终评分 评估脚本将独立计算所有5个任务的个人准确度分数。最终排行榜排名严格由**平均分数**决定，即所有任务类别分数的平均值。
